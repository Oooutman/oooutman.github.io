Vultr 新人免费250美金免费试用教程

前言

我自己第一次买 VPS 服务器就是 Vultr 家的，当时也没有新人免费试用活动。选 Vultr 的理由主要是看中它按时间收费和免费更换 IP 两点，比如你买了个 3 美元/月的实例，新手买了不确定自己是否能用的来，可能第一天就受挫折不想用了，这时你可以删除实例，钱只会扣除你这段时间的费用，可以等你哪天调整好状态再新开个实例来学习使用。如果 VPS 的 IP 被墙了，你想搭建科学上网也是不行的，Vultr 可以新建实例重新分配个 IP，然后再删除被墙 IP 的实例，这样相当于免费更换 IP。

当然现在有新人白嫖活动，以上的当心都是多余，不过要注意赠送的 250 美元只有 1 个月时效，老客户别冒险尝试，可能会导致老帐号也被封了，以后你的 IP 想注册都注册不了。给新人的建议是多尝试，比如你要搭建代理翻墙工具，那么可以各个地区的节点都尝试下，没必要尝试很贵的，最低档的基本就满足需求量。还可以试试块存储搭建网盘离线下载，或者尝试搭建 WordPress 或 Hexo 博客，普通人用的最多也就这几点了。掌握一些技术后以后付费购买自己心里就更有底了，毕竟账户最低充值 10 美元门槛还是高了点。

注册账号

点击这个链接进入赠金账号注册页面 https://www.vultr.com/promo/try250

刚点击链接会先出现一个安全验证页面判断是否是真人，选框打勾好了，然后就转到赠金账号注册了（这里我使用了浏览器网页翻译，看不懂英文也可以用网页翻译，注意有时在提交确认选项的时候翻译后的网页会失效），按要求填写邮件地址和密码，注意密码的规范，好像会给你发邮件确认，记不太清了，注意看提示。

获得 250 美元赠金

注册完成登陆后，按下图顺序点击 Billing（计费） 选项，再点击 Gift Code（礼品码） ，输入礼品码：FLYVULTR250 后提交就行，这样赠金就到账了。

实例部署

点击左侧的 Products（产品） 选项，鼠标移动到蓝色的圆形带+号的图标上，再点击第一个下拉选项 Deploy New Server（部署新服务器），

进入新页面后第一项是选择服务器，这里选第二个 云计算 。处理器和存储这里四个选项一般选第四个 intel常规 ，因为更便宜，不过有赠金的话选前面两个也可以。

服务器位置当然是离我们越近越好，这样速度快延迟也低，东京、汉城、新加坡 这三个节点比较推荐。如果自费想省钱，在上一步处理器选择项选了第四个的，选择 纽约 节点等下会有个更便宜的选项。

操作系统选第一个 AlmaLinux，点击一下选择 8 x64 版本，相对来说较最新版本，前一个版本BUG更少，稳定性也更好。下面的套餐我选了 3.5 美元的，如上所述，只有处理器项选第四个和纽约节点才有3.5美元，2.5美元的没有 IPv4 别选。如果处理器项选第四个其它节点最低也要5美元。

TIP
除了最后两个 Windows 系统，其它都是不同 Linux 的发行版，你可以理解为不同安卓手机品牌用的都是安卓系统，但是又有所区别。这些发行版大致又可分为 RedHat（红帽） 和 Debian 两派，对于新人来说它们之间最主要的区别就是包管理器和软件仓库，体现在使用中的区别就是安装软件工具和升级系统时使用的命令不同，比如要安装 Vim， RHEL 的命令是 yum install vim，Debian 的命令则是 apt-get install vim。

这里我们安装的 AlmaLinux 属于 RedHat 派系，找各种搭建教程时要看清人家的安装命令是不是你能用的。

去掉自动备份的选项，弹出提示打勾禁用就行了，然后最下面的主机名和标签随便填下就可以了，确认下价格就可以点击部署了。

最后

跳转到你的实例页面稍微等下，等状态变为绿色 Running，然后鼠标移到 IP 上，会出现复制按钮跟提示，先复制 IP ，看下能不能 ping 通。按下快捷键 Win+R 打开运行，输入 cmd 确定，因为我的 IP 是 45.63.22.166 ，于是输入 ping 45.63.22.166 回车。如果有延时显示就是通的能连上，提示超时就是这个 IP 被墙了连不上。你先别删除实例，按前面的步骤重新申请一个，等状态绿了再删掉被墙的这个，不然先删再申请很容易给你同个 IP 。

确保能 ping 通后，想登录服务器我们还得获取到服务器的相关信息，鼠标移到国旗上点下，页面跳转到服务器信息，IP 地址、用户名、密码这三项是我们登录服务器的关键信息。
