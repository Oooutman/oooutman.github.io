科学上网 VPS 推荐 (Vultr 购买使用教程)

前言

Vultr 家有两个优点：一是按时间收费，购买虽然按月购买，不过不用可以随时删了停止计费，用半个月删了只扣一半费用（必须删除实例才停止计费，只是关机还是会继续扣费）。第二个优点是 IP 免费更换，如果 VPS 的 IP 被墙了，你可以先将给当前机子创建个快照，然后重新购买机子使用快照部署，完成后再删除 IP 被墙的机子，这样就相当于免费更换 IP 了，也方便根据需求随时切换地区，比如我想免费看环法我会使用一段时间法国地区节点。

Vultr 的缺点应该是最低充值 10 美元的门槛，带 IPv4 的机子最低月付只要 3.5 美元（限纽约地区，其它地区最低 5 美元起）。还有缺点是它家 IP 用不了微软 Copilot 系列的 AI 工具，ChatGPT 和谷歌的 Gemini 正常。Netflix 之类付费流媒体基本也不行，YouTube 这类流媒没问题。

注册账号

点击这个活动链接开始注册 https://www.vultr.com/match/，下图可直接在选框内填写信息注册账号。
目前新用户通过这个链接注册有充值金额翻倍活动（充 $10 就是送 $10，最高充 $100 送 $100，仅限一次），赠送的金额有效期一年，每次购买服务扣费时会根据订单金额从赠金和本金各扣一半。

注册完后可能会让你填写个人资料才让购买，记不太清，没提示就不用管，有要求就填下，除了邮箱其它都可以随便乱填，但不建议使用国外的信息，地址你就填本市随便哪个街道就行，记得中文翻译成英文。

充值

然后按下图顺序点击 Account（账户） 和Make a Payment（进行付款） ，选择 Alipay（支付宝） 充值。

购买实例部署

点击左侧的 Products（产品） 选项，再点击右上角 Deploy +（部署）按钮跳转购买。

再如下图选择 VPS 类型和地区，注意只有纽约(新泽西)这个节点有 $3.5 的机子，其它地区最低 $5，不差这点钱推荐选亚洲地区的机子，近点网络会好点。

操作系统这里选 Debian 12 x64 ，因为很多针对初学者的教程还有一键脚本 Debian 用的比较多，这样用的会顺点。

除了后面两个 Windows 系统，其它都是 Linux 不同的发行版，你可以理解为不同安卓手机品牌用的都是安卓系统，但是又有所区别。常见两大派系分为 RedHat（红帽） 和 Debian 两派，对于新人来说它们之间最主要的区别就是包管理器和软件仓库，体现在使用中的区别就是安装软件工具和升级系统时使用的命令不同（可以理解为从应用商店下载软件用到的命令）。比如安装 Vim， RedHat 的命令是 yum install vim，Debian 的命令则是 apt-get install vim。找各种搭建教程时要看清人家的安装命令是不是你系统能用的。

套餐类型选 Regular Cloud Compute(常规云计算)，使用上代 CPU 和 SATA SSD 会便宜点，对于科学上网够用了。

附加功能这里取消自动备份，弹出提示打勾禁用就行了。

最后随便填下主机名和标签就可以了，确认下价格没错就可以点击部署了。

获取服务器登录信息

部署后得稍微等下，等状态变为绿色 Running，再照下图查看服务器详细信息

右下角红框内的 IP 地址、用户名和密码就是登录服务器的关键信息，不要外泄，这三个信息边上有个复制的图标。

在此之前先 ping 下 IP ，看下能不能 ping 通。PC 用户按下快捷键 Win+R 打开运行，输入 cmd 确定，假如我的 IP 是 88.60.22.166 ，就输入 ping 88.60.22.166 回车。如果能看到延时就是能连上，如果服务器状态 Running 了两三分钟，ping 却超时就是这个 IP 被墙了连不上。你先别删除实例，按前面的步骤重新申请一个，等状态绿了再删掉被墙的这个，不然先删再申请很容易给你同个 IP 。

如果服务器运行一段时间出现 IP 被墙，这时机子里已经安装了一些的东西不想重装一遍。你可以在服务器详细信息里创建个 快照（Snapshot），然后新建实例选择系统时切换到 Snapshot 菜单，选择你创建的快照部署就行了。
