虽然我很反感宝塔的一些行为，不过还是得承认这种可视化面板能降低小白的学习门槛，帮助理解并减少错误发生。
下面是宝塔 7.7 旧版本的安装脚本，如果不介意手机注册绑定可以去宝塔官网下载新版。
```
curl -sSO https://raw.githubusercontent.com/8838/btpanel-v7.7.0/main/install/install_panel.sh && bash install_panel.sh
```

运行命令后会有如下提示问你是否确认安装，输入 `y` 回车就开始安装了。
```
Do you want to install Bt-Panel to the /www directory now?(y/n): y
```

等个几分钟（我这里用时 5 分钟），出现如下提示就是安装成功了，可以复制 `外网面板地址` 到浏览器打开（面板地址可以保存下书签方便下次登录），使用 `username`（用户名）和 `password`（密码）登录。不过先别急着登录，下面还有一些操作。
```
==================================================================
Congratulations! Installed successfully!
==================================================================
外网面板地址: http://35.197.113.2:8888/1ea3bf4e
内网面板地址: http://10.138.0.3:8888/1ea3bf4e
username: odopay41
password: db21e3af
If you cannot access the panel,
release the following panel port [8888] in the security group
若无法访问面板，请检查防火墙/安全组是否有放行面板[8888]端口
==================================================================
Time consumed: 5 Minute!
```
因为面板使用的 8888 端口，使用以下命令防火墙放行下 8888 端口，否则网页打不开。第二行还放行了一个 6800 端口，给之后的要安装的 Aria2 使用的。
```
iptables -I INPUT -p tcp --dport 8888 -j ACCEPT
iptables -I INPUT -p tcp --dport 6800 -j ACCEPT
```
最后使用下面两个命令屏蔽和删除绑定手机号的一些设置，完事就可以登录面板了。
```
sed -i "s|bind_user == 'True'|bind_user == 'XXXX'|" /www/server/panel/BTPanel/static/js/index.js
rm -f /www/server/panel/data/bind.pl
```
登陆后可以先进入 `面板设置` 修改下用户名和密码方便记忆，如果你忘了用户名密码甚至是面板的登录地址，可以在 SSH 登录服务器后输入 `bt` 命令，可以打开宝塔的管理界面，输入 `14` 可以查看面板的默认信息。
